# Task 4 - Power BI Dashboard

## Objective
Design an interactive business dashboard using Power BI.

## Dataset
Superstore Sales Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales Trend Line Chart
- Sales by Category
- Profit by Region
- Sales by Segment
- Sales by Sub-Category
- Interactive Slicers

## Tools Used
- Power BI Desktop
- Microsoft PowerPoint
- GitHub

## Outcome
Created an interactive dashboard to analyze sales performance and support business decision-making.
